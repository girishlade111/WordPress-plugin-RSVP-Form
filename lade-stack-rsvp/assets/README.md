## Asset Guidelines

### Banners
- Use high-contrast design
- Include plugin name clearly
- Match brand colors (#667eea primary)
- No sensitive information

### Icons
- Simple, recognizable design
- Works at small sizes
- Transparent background (PNG)
- Square aspect ratio

### Screenshots
- Show key features
- High resolution (minimum 1280x720)
- Include captions in README.txt
- PNG format for best quality

## Design Specifications

### Color Palette
- Primary: #667eea (Purple)
- Success: #48bb78 (Green)
- Warning: #ed8936 (Orange)
- Danger: #f56565 (Red)
- Background: #e0e5ec (Light Gray)

### Typography
- Use modern sans-serif fonts
- Minimum 16px for readability
- High contrast for accessibility

### Branding
- Include "Lade Stack" name
- Use neumorphic design elements
- Consistent with plugin UI

## WordPress.org Requirements

From [WordPress Plugin Handbook](https://developer.wordpress.org/plugins/wordpress-org/plugin-assets/):

- Banners are optional but recommended
- Icons are optional but recommended
- At least one screenshot required
- All assets must be GPL-compatible
- No trademarked elements unless you own them
