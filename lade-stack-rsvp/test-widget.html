🚀 Setup Instructions

Step 1: Install Plugin

  1. Upload the lade-stack-rsvp folder to /wp-content/plugins/
  2. Activate "Lade Stack RSVP Widget" in WordPress admin
  3. Add shortcode to any page or post

Step 2: Configure EmailJS (Optional)

  1. Sign up at EmailJS.com (free)
  2. Add an Email Service (Gmail, Outlook, etc.)
  3. Create an Email Template with these variables:
    {{name}}
    {{email}}
    {{event_name}}
    {{event_date}}
    {{event_time}}
    {{event_location}}
    {{guests}}
    {{dietary}}
  4. Copy Service ID, Template ID, and Public Key to shortcode

Step 3: Test the Widget

  1. Drag: Grab the header (⋮⋮ icon) to move the widget
  2. Resize: Drag the bottom-right corner
  3. Fill Form: Enter name, email, and other details
  4. Submit: Click "Reserve My Spot" to see confirmation
  5. QR Code: Download your unique QR code
  6. Admin Dashboard: Click ⚙️ button (password: admin)
  7. Export CSV: Download all RSVPs from dashboard
💡 Admin Password Hint: The default password is admin. Change it in the shortcode: admin-password="yourpassword"
